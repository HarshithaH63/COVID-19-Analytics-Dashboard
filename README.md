## Project Overview

This project presents an interactive **COVID-19 analytical dashboard** built using Power BI.
The dashboard visualizes global pandemic data and helps users understand infection trends, recovery patterns, and country-level impact through interactive charts, maps, and performance indicators.

The goal of this project is to transform raw COVID-19 data into meaningful insights using data visualization and reporting techniques.

---

## Tool Used

* Power BI

---

## Dashboard Features

### Key Performance Indicators (KPI Cards)

The dashboard includes summary cards showing:

* Total Confirmed Cases
* Total Deaths
* Total Recovered Cases
* Total Active Cases
* Case Fatality Rate
* Recovery Rate

---

### Filters (Slicers)

Users can interact with the dashboard using filters:

* Country
* Year
* Month
* Date

These filters allow dynamic exploration of pandemic trends over time and across regions.

---

## üìà Visualizations

### COVID-19 Trends

* Monthly and yearly trends of confirmed, deaths, and recovered cases
* Allows users to observe how the pandemic evolved over time

### Top Impacted Countries

* Top 5 countries with the highest fatality rate
* Enables comparison of outbreak severity between countries

### Recovery Analysis

* Top 5 regions with the highest recovery rate
* Helps identify regions with better recovery outcomes

### Regional Case Distribution

* Cases by region visualization for comparative analysis
* Shows which regions were most affected

### Map Visualization

* Interactive global map displaying total cases by region
* Helps visualize geographic spread of COVID-19 worldwide

---

## Objectives

* Monitor pandemic spread across countries and regions
* Compare fatality and recovery rates
* Identify highly affected regions
* Provide clear, interactive visual insights from real-world data

## How to Use

1. Download the `.pbix` file from this repository
2. Open the file using Power BI Desktop
3. Use filters (country, date, month, year) to interact with the dashboard
